Logging in to MathLab

Software and documentation

  • MacOS comes with the application "terminal". Simply click to run it. Login to MathLab by typing ssh utorid@mathlab.utsc.utoronto.ca Enter your password

  • On Windows, you will need an SSH client. PuTTY is a good free one. You can download and install it here: PuTTY. Use the same command as above to login to mathlab: ssh utorid@mathlab.utsc.utoronto.ca
